# AI客服插件

基于大语言模型的智能客服插件，帮助网站实现自动化客户服务。

## 功能特性

- 🤖 智能对话 - 基于AI大模型，理解用户问题
- 🌐 多模型支持 - 豆包(免费)、千问(付费)
- 📦 简单易用 - 复制粘贴即可使用
- 💼 适配任何网站

## 版本

### 基础版
- AI对话功能
- 免费使用

### 专业版
- AI对话 + 知识库问答
- WordPress插件支持
- 付费使用

## 快速开始

1. 下载 `AI客服-基础版.html`
2. 修改API Key
3. 上传到网站

详见 [使用教程](./AI客服-使用教程.md)

## 技术栈

- HTML5 + CSS3 + JavaScript
- 豆包API / 千问API
